# dorasite_archives
消えゆくサイト保存のために: タイトルに対応するサイトはローカルに階層ごと保存済みです


## 保存方法
外部サイトへの参照を除く場合は`-H`オプションはつけない. 

```
wget -rkp -linf -H [target site domain]
```
